Experience and Portrofilo

Summary of My experience and portrofilo:

The project I was involved in was a redesign of a smart home control system to improve its usability, functionality, and aesthetic appeal.
The original system had a complex interface and lacked intuitive controls, leading to frustration for users.
The goal was to create a streamlined and visually pleasing user experience while maintaining seamless integration with existing smart devices.
The design process involved user research, user-centred design, information architecture restructure, wireframing and prototyping, and visual design.
The redesigned system offered homeowners an intuitive interface with clear controls for lighting, HVAC, security, and entertainment systems.
The visual design conveyed a sense of simplicity and elegance, seamlessly integrating with the home's interior design.
The project resulted in increased user satisfaction, reduced frustration, and improved efficiency in controlling and monitoring home systems.
Homeowners appreciated the streamlined interface and found it easy to customize settings, create automation routines, and monitor energy usage.
This project showcased the team's expertise in user-centred design, information architecture, wireframing, prototyping, visual design, and usability testing.
Another project I was involved in was designing an interactive educational software interface for elementary school students.
The goal was to create a visually appealing and engaging interface that would encourage participation and enhance their learning experience.
The design approach involved a vibrant color palette, custom illustrations, clear typography, a clean and intuitive layout, visual feedback, gamification elements, and responsive design.
The color palette was chosen to evoke fun and excitement while maintaining visual harmony.
Custom illustrations and icons were used to make the interface relatable to young students, creating a sense of familiarity and engagement.
A clear and legible typeface was selected for young readers, with font variations such as bold and italic emphasized.
The interface was organized using a grid-based structure, with clear visual cues like buttons and arrows.
Visual feedback was provided through interactive buttons that changed color or displayed animated effects upon clicking.
Gamification elements were integrated to track students' achievements and provide a sense of accomplishment.
The interface was responsive and adaptable across various screen sizes, ensuring visual appeal and usability on different devices.
The project exemplifies how visual appeal and design principles can be applied to create an interactive educational interface that captivates and inspires young learners.
